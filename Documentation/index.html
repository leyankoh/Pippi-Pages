Cats - A base model

Cat objects can be instantiated like so:


const Pippi = new Cat();

To add features to your cat, you can declare the following object properties:


Pippi.furColour = "Tuxedo";

Things Specific to the Pippi edition cat

New Pippi objects differ greatly from base Cat objects. Hence we practice inheritance from Cat objects then add in properties specific to only Pippi.


class Pippi(Cat):
    Pass

Example of Pippi instantiator:


class Pippi(Cat):     def __init__(self, catName):         #add properties etc.

Pippi objects also contain the following properties:

  • Noise
  • Turkey Kibble Level
  • Poop Level
